FingerprintEngine.python
движок обработки видео и получения индексов/метрик

файлы:

1) main.py 
главный файл движка

2) все файлы проекта
начинаются с "alpha_"

3) библиотека ffmpeg (файлы из стандартной установки бинарников):
ffmpeg.exe - для загрузки файла в памяти
ffprobe.exe - для получения информации о файле (длина, размер кадра...)
ffplay.exe  (пока не используется)
должны быть в корне проекта, чтобы их смогла вызвать библиотека ffmpeg-python

4) библиотека openCV (файлы скачаны из git-репозитория):
все, что начинается с "haarcascade_" и заканчивается на ".xml"
конфигурационные файлы для разных классификаторов
всего 17 шт
должны быть в корне проекта, чтобы их смогла вызвать библиотека opencv-python
